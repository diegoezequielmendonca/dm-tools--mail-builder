# Mail Builder

Crea un Newsletter rápidamente a partir de una lista de componentes pre-armados y descarga el código HTML resultante

## Instalación

_Es necesario tener NodeJS_

- Guardar los componentes HTML en la carpeta **/components**
- Ejecutar el commando ```npm start``` en el root de la aplicación
	- Node creará el archivo **components.json** dentro de la carpeta **/assets/js**

## Documentación

### Modo de uso

_Es necesario montar un servidor local_

- Acceder a la aplicación desde el servidor local
- Los nombres de los componentes figurarán en el menú izquierdo
- Hacer click en los componentes deseados para agregarlos al Newsletter
- Situar el mouse sobre un componente del Newsletter para ver las opciones
	- Es posible borrar o mover el componente desde las opciones
- Presionar el botón **Cancelar** para borrar el contenido
- Presionar el botón **Descargar** para guardar el Newsletter generado con la aplicación
